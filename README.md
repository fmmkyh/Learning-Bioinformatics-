# Dokumentasi Proyek Translriptomik - Bioinformatics Research Starter Program
Proses belajar kurang lebih berjalan sebulan dan dilaksanakan melalui Google Classroom.
  Week 1: Memahami dan merangkum paper berbahasa Inggris terkait _transcriptomics_.
  Week 2: Analisis DEG menggunakan GEO2R terhadap dataset  _transcriptomics_ publik.
  Week 3: Analisis kasus dengan pemilihan data, metode analisis, dan interpretasi hasil.
Adapun proyek akhir berupa penyusunan analisis secara mandiri, mulai dari perumusan masalah, 
interpretasi hasil, dan dokumentasi akhir
