# Dokumentasi Proyek Transkriptomik - Bioinformatics Research Starter Program
<p>Proses belajar kurang lebih berjalan sebulan dan dilaksanakan melalui Google Classroom.</p>
<ol>  
  <li>Week 1: Memahami dan merangkum paper berbahasa Inggris terkait <m>transcriptomics</m>.</li>
  <li>Week 2: Analisis DEG menggunakan GEO2R terhadap dataset <m>transcriptomics</m> publik.</li>
  <li>Week 3: Analisis kasus dengan pemilihan data, metode analisis, dan interpretasi hasil. </li>
</ol>
<p>Adapun proyek akhir berupa penyusunan analisis secara mandiri, mulai dari perumusan masalah, interpretasi hasil, dan dokumentasi akhir.</p>
